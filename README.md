# Discord Betting Bot

Bot de apostas para Discord integrado com Mercado Pago PIX.

## Funcionalidades

- Sistema de apostas 1v1, 2v2, 3v3 e 4v4
- Criação automática de salas privadas
- Integração PIX Mercado Pago
- Confirmação de jogadores e moderadores
- Timeout automático
- Logs automáticos
- Sistema de vencedor

## Tecnologias

- Python
- discord.py
- Mercado Pago SDK

## Instalação

Clone o projeto:

```bash
git clone https://github.com/SEUUSUARIO/Bot-api-pix.git
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

## Configuração

Crie um arquivo `.env`

```env
TOKEN=
MP_ACCESS_TOKEN=
```

## Executar

```bash
python bot.py
```
